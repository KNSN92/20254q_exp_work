# プロダクト開発: 10年後のZEN Study

## コンセプト
- 個別AI教師
- スマートグラス

## 担当別ワーク

### 概要・ターゲット・ビジネスモデル
#### 概要
- 現在ではVRゴーグルを使用した学びが可能となっているが、重さやVR酔いが課題になっている。そこで現在meta社が取り組んでいるスマートグラスを使用した本当にいつでもどこでも学べる環境を提供
- 個別AI教師の導入により、24時間いつでも教師に質問できる安心感を提供
- 音声操作を可能にし、言語の科目では発音から詳しく学ぶことが可能

#### ターゲット
- メイン　15~18
- サブ１　12~15
- サブ２　20代

-

### 画面の構成・UI
- ジェスチャや視点でUI操作
- 画面上に仮想的なキーボードを出現させて手の動きで入力出来るようにする
- 手ぶれでも操作しやすい大きなインターフェース
- 実際の視界も見れるように半透明なUIにする

### 機能一覧
- スマートグラス連動学習
- 個別AI教師機能
- 計算式や文章などが画面上に浮かび上がっており、書き加える事も出来る
- AIのアバターが表示されており、身振り手振りで指導してくれる
- 音声入力を使用して英会話の練習なども可能
